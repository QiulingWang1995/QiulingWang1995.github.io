在Github上用Markdown创建技术博客

实验一 在Github上用Markdown创建技术博客

一、实验目的

1.熟悉Github网站

2.掌握Markdown语法

二、实验任务

1.注册Github账号

我的账号：QiulingWang1995

2.创建Github Pages repository

3.发布第一篇博客


4.学习Markdown

5.利用hexo生成博客并部署到github

三、实验步骤与结果

一、本地配置hexo

1 安装node-v7.7.1-x86.msi

2. 安装Git-2.12.0-32-bit

3. 设置npm镜像为淘宝，http://npm.taobao.org/

1 cnpm install -g cnpm –registry=https://registry.npm.taobao.org

4. 安装Hexo

1 cnpm install –g hexo -cli

5. 新建文件夹MD到D盘根目录，复制地址D：/MD

1 cd /d D：/MD

6.初始化站点

1 hexo init hexo
2 cd hexo
3 cnpm install

7.浏览本地站点

1 hexo server

二、在GitHub上部署Hexo

1.创建一个Github Pages repository

2.修改配置文件——_config.yml（Hexo文件根目录下）

1 type: git
2 reposutoty: https://github.com/QiulingWang1995/QiulingWang1995.github.io.git
3 branch: master

3.配置site部分（可选）

1 # Site
2 title: 汪秋凌的博客
3 subtitle: 自建博客步骤
4 description:
5 author: 汪秋凌
6 language:
7 timezone:

4. 如要要修改主题，将主题文件克隆（clone）到themes目录下，并修改_config.yml配置文件中themes参数（如landscape改为jacman）。

5. 安装 hexo-deployer-git

1 npm install hexo-deployer-git –save

6.配置git全局邮箱与名字

1 git config –-global user.email “ql.wang@gmail.com”
2 git config –-global user.name “Qiuling Wang”

7.生成博客

1 hexo generate

8.部署博客

1 hexo server

三、添加新文章

1.生成新文章

1 hexo new newblog

2.用编辑器修改文章

2 myhexo\source_posts\newblog.md

3.生成并发布到Github

3 hexo d –g

四、实验小结

通过学习搭建博客平台，对博客平台和Markdown开发文档有了基础的了解。
